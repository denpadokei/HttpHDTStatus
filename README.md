# HttpHDTStatus
HMDの移動距離をオーバーレイで表示するためのMod
# 依存Mod
- HttpSiraStatus
- SiraUtil
- HeadDistanceTravelled
# パラメーター
Otherとして飛んできます。
```json
{
    "HeadDistanceTravelled": {
        "Posoition": {
            "x": 0.0,
            "y": 0.0,
            "z": 0.0
        },
        "Rotation": {
            "x": 0.0,
            "y": 0.0,
            "z": 0.0
        },
        "Distance": 0.0
    }
}
```
